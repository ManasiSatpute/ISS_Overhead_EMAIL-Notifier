# ISS Overhead Email Notifier 

A Python automation project that tracks the International Space Station (ISS) in real time and sends an email alert when it passes overhead during nighttime.

##  Features
- Real-time ISS location tracking
- Nighttime detection using Sunrise-Sunset API
- Email notification using SMTP
- Secure credentials using environment variables
